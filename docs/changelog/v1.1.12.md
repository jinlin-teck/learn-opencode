# v1.1.12：组织订阅支持

**发布日期**：2026年1月10日

---

## 一句话总结

这次更新让 **OpenAI 组织订阅用户** 可以正常使用 OpenCode，解决了之前组织账户无法正确识别的问题。

---

## 问题修复：OpenAI 组织订阅支持

### 之前的问题

如果你的 OpenAI 账户属于某个**组织（Organization）**，比如公司购买的 API 订阅，之前可能会遇到：

- API 调用失败
- 账单计入错误的账户
- 无法使用组织的配额

### 现在的情况

OpenCode 现在会自动发送 `ChatGPT-Account-Id` header，确保：

- ✅ API 调用正确关联到你的组织账户
- ✅ 费用计入组织账单
- ✅ 使用组织的 API 配额

### 谁需要关注？

| 情况 | 影响 |
|------|------|
| 个人 OpenAI 账户 | 无影响，正常使用 |
| 公司/组织 OpenAI 账户 | ✅ 修复了之前的问题 |
| 使用其他模型（DeepSeek、Claude 等） | 无影响 |

---

## 要不要升级？

| 情况 | 建议 |
|------|------|
| 使用公司/组织的 OpenAI 账户 | ✅ 必须升级 |
| 个人 OpenAI 账户 | 可以升级，无副作用 |
| 不用 OpenAI | 可以等下次一起升级 |

**升级命令**：

```bash
opencode upgrade
```
