# v1.1.64：可关闭音效

**发布日期**：2026年2月12日

---

## 一句话总结

这次更新让 **桌面端可以关闭音效**，修复 Windows 选择行为，改进 CLI 模型回退。

---

## 新功能：关闭音效

### 这次更新了什么？

桌面应用现在可以 **关闭音效**，如果你不喜欢提示音，可以在设置中关闭。

### 如何设置？

在桌面应用的设置中找到音效选项，关闭即可。

---

## 改进：Windows 选择行为

### 这次更新了什么？

修复了 Windows 上的选择行为，并添加了手动 Ctrl+C 处理。

### 好处是什么？

- 文本选择更正常
- Ctrl+C 行为更一致
- Windows 体验更好

---

## 改进：CLI 模型回退

### 这次更新了什么？

在 CLI 中查找最近使用的模型时，回退逻辑更合理。

---

## 改进：基线 CPU 检测

### 之前的问题

基线 CPU 检测有时不准确。

### 现在的情况

修复了基线 CPU 检测问题，二进制文件选择更准确。

---

## 改进：终端 PTY 隔离

为应用添加了终端 PTY 隔离，避免不同会话之间的干扰。

---

## 要不要升级？

| 情况 | 建议 |
|------|------|
| 不喜欢音效 | ✅ 升级 |
| Windows 用户 | ✅ 升级 |
| 用得好好的 | 可以等下次 |

**升级命令**：

```bash
opencode upgrade
```
