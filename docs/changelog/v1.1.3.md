# v1.1.3：Bedrock 增强 + 桌面版 CLI 安装

**发布日期**：2026年1月5日

---

## 一句话总结

这次更新增强了 **Amazon Bedrock 支持**，桌面版可以**一键安装 CLI**，还修复了多个稳定性问题。

---

## 新功能：Bedrock 配置增强

### 这次更新了什么？

Amazon Bedrock 用户现在有更多配置选项：

- 支持 **jp. 前缀**的跨区域推理配置
- 更清晰的**认证优先级**
- 更灵活的配置方式

### 认证优先级

1. 配置文件中的 API Key
2. 环境变量
3. AWS CLI 配置
4. IAM 角色

👉 **相关教程**：[模型提供商](/appendix/providers)

---

## 桌面版：一键安装 CLI

### 这次更新了什么？

桌面版现在可以**一键安装命令行工具**，不用单独下载安装了。

### 如何使用？

1. 打开桌面版设置
2. 找到「安装 CLI」选项
3. 点击安装

安装后就可以在终端使用 `opencode` 命令了。

---

## 改进：权限配置顺序保留

### 之前的问题

在配置权限规则时，规则的顺序可能会被打乱，导致优先级不符合预期。

### 现在的情况

现在会**保留你定义的顺序**，第一条规则优先级最高。

---

## 改进：自定义命令支持图片

### 之前的问题

使用自定义命令（Custom Commands）时，无法传递图片参数。

### 现在的情况

现在可以在自定义命令中使用图片了，比如截图分析、图片处理等场景。

---

## 改进：Tab 展开目录

### 之前的问题

在文件自动补全中，按 Tab 和 Enter 的行为不够直观。

### 现在的情况

- **Tab**：展开目录，查看子文件
- **Enter**：选中当前项

更符合直觉的操作方式。
